# Customer-Churn-Prediction
A machine learning project that analyses customer data to predict churn behaviour. The project includes data preprocessing, exploratory data analysis, feature engineering, model development, and evaluation of classification algorithms to identify factors influencing customer retention.

# Customer Churn Prediction

## Project Overview
This project focuses on predicting customer churn using machine learning classification techniques. The goal is to identify customers who are likely to leave a service and understand the key factors influencing churn.

## Dataset
The project uses a customer churn dataset containing customer demographic information, account details, and service usage information.

## Project Workflow
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- Feature engineering
- Data scaling and preparation
- Machine learning model development
- Model evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## Models Used
- Logistic Regression
- Random Forest

## Results
The models were evaluated using accuracy, precision, recall, F1-score, and other classification metrics to determine predictive performance.
